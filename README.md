# VA_Disability_Calculator
User Interface for Calculating VA Disability Rates and Payments

# main.py
Main file for the application; includes the App class. Run this file to run the application.

# frames.py
Contains the four frame classes used to build the application.

# widgets.py
Contains the customized widgets used within the frames of the application.

# settings.py
Contains global variables for the width and height of the application and the Fiscal Year 2023 pay charts used to calculate the VA payment.

# va_rating.py
Contains the VA_Rating class used to calculate the combined rating and the monthly payment.
